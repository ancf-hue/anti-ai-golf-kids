  # 🏌️ Антит ИИ Гольф 3D

Безопасная, развивающая 3D-игра в гольф, созданная специально для детей. 

В эпоху цифровых манипуляций этот проект предлагает чистый, честный игровой опыт: настоящая физика, реальные навыки и никаких алгоритмов, подстраивающихся под игрока. Игра развивает пространственное мышление, мелкую моторику и учит планировать свои действия.

🔗 **[Играть онлайн прямо сейчас](https://ВАШ_НИК.github.io/anti-ai-golf-3d/)** *(замените ссылку на вашу после публикации)*

## ✨ Особенности
- 🧒 **Безопасно для детей:** Мягкая, приятная озвучка без резких или пугающих звуков.
- 📱 **Работает везде:** Адаптирована для ПК, планшетов и смартфонов (поддержка сенсорного управления).
- 🚫 **Никакой рекламы и трекинга:** Полностью автономный HTML-файл, который можно запустить даже без интернета.
- 🎨 **Современный дизайн:** Интерфейс в стиле Glassmorphism (эффект матового стекла).
- 🏆 **10 уровней:** От простых прямых ударов до сложных лабиринтов с препятствиями (вода, песок, стены).

## 🎮 Как играть
1. **Удар:** Зажмите левую кнопку мыши (или палец на экране) на мяче и потяните в сторону, противоположную желаемому направлению удара.
2. **Сила удара:** Чем дальше вы тянете, тем сильнее будет удар (следите за шкалой силы внизу).
3. **Камера:** Кликните по пустому полю и потяните, чтобы вращать камеру вокруг уровня. Используйте колесико мыши для зума.
4. **Цель:** Загнать мяч в лунку за минимальное количество ударов.

## 🛠 Технологии
Проект создан с использованием минимального набора технологий для максимальной скорости и совместимости:
- **HTML5 / CSS3** (Glassmorphism UI, адаптивная верстка)
- **Vanilla JavaScript** (логика игры, физика, обработка касаний)
- **Three.js (r128)** (легковесный 3D-рендеринг через WebGL)
- **Web Audio API** (программная генерация мягких звуков без внешних аудиофайлов)

## 🚀 Как запустить локально
1. Скачайте или склонируйте этот репозиторий.
2. Поместите файл `favicon.ico` в ту же папку, где лежит `index.html` (по желанию).
3. Просто откройте файл `index.html` двойным кликом в любом современном браузере (Chrome, Firefox, Safari, Edge).
4. Установка сервера или Node.js **не требуется**.
